## 觀念題與名詞解釋:精簡 摘要說明底下觀念 [請說明重要性]
```
1.	Von Neumann architecture
 - 范紐曼型架構
 - 將程式指令記憶體和資料記憶體合併在一起的電腦設計概念架構
 - 是一種相對於平行計算的序列式架構參考模型
 - 是一種實作通用圖靈機的計算裝置
 - 以運算單元為中心
2.	instruction cycle
 - 又稱提取－執行周期
 - 是執行一條指令所需要的時間
 - 一共是4步驟  - 取得指令 - 解碼指令- 執行指令 儲存結果
 - 指 CPU 要執行一條機器指令經過的步驟
 - 由若干機器周期組成
3.	HTML
 - 是一種用於建立網頁的標準標記語言
 - 是打造網頁的基石
 - 它表述並定義網頁的內容
 - 常與 CSS 與 JavaScript 一起使用在網頁設計上
 - 超文本標記語言 (Hyper Text Markup Language)
4.	CSS
 - 是一種風格頁面語言
5.	Javascript
6.	RWD
```
